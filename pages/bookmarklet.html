Bookmarklet

Instala el script en tu navegador para exportar tickets de ServiceNow

1

Arrastra el botón a tu barra de favoritos

Arrastra el siguiente enlace a tu barra de marcadores del navegador. Asegúrate de que la barra esté visible (Ctrl+Shift+B en Chrome/Edge).

Arrastra hacia la barra de favoritos
Export Ticket
 

Si no puedes arrastrar, haz clic en "Copiar código" y crea un marcador manualmente pegando el código como URL.

2

Úsalo en ServiceNow

Abre cualquier ticket en ServiceNow y haz clic en el marcador Export Ticket que acabas de guardar.

  1. Navega al ticket en ServiceNow (INC, RITM, SCTASK…)
  2. Haz clic en Export Ticket en tu barra de favoritos
  3. Aparecerá un botón azul "Export Ticket" en la página
  4. Haz clic en ese botón — el JSON se copia al portapapeles
  5. Ve a Form en Agent Eval y pega el JSON
3

Pega en Agent Eval

En la página Form, usa el botón "Paste ServiceNow" o pega directamente en el campo de importación. El formulario se completará automáticamente.

Ejemplo de datos exportados
{
  "number": "INC0435166",
  "shortDescription": "Cannot access shared drive",
  "assignedTo": "John Smith",
  "assignmentGroup": "ITEC-SUP PS",
  "state": "In Progress",
  "slaBreach": "No",
  "slaPercentage": 72.3,
  "slaName": "ITEC-SUP PS - Medium Priority Incident"
}

Solución de problemas

El botón "Export Ticket" no aparece en ServiceNow

Asegúrate de estar dentro de un ticket abierto. El script necesita que g_form esté disponible en la página. Si usas ServiceNow con vista clásica, el script también funciona.

slaPercentage aparece como null

Expande la sección "SLA" / "Related Lists" en el ticket antes de hacer clic en Export Ticket. ServiceNow carga las SLAs de forma diferida.

Error: "g_form no disponible"

Este mensaje aparece si haces clic en el marcador fuera de un ticket (ej. en la lista de incidentes). Abre un ticket específico y vuelve a intentarlo.

No puedo arrastrar el marcador (Safari / Firefox)

Haz clic en "Copiar código del bookmarklet", luego crea un nuevo marcador manualmente y pega el código como URL del marcador.
