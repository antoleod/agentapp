Bookmarklet

Install the script in your browser to export ServiceNow tickets

1

Drag the button to your bookmarks bar

Drag the link below to your browser's bookmarks bar. Make sure the bar is visible (Ctrl+Shift+B on Chrome/Edge).

Drag to the bookmarks bar
Export Ticket
 

If you can't drag it, click "Copy bookmarklet code" and create a bookmark manually by pasting the code as the URL.

2

Use it in ServiceNow

Open any ticket in ServiceNow and click the Export Ticket bookmark you just saved.

  1. Navigate to the ticket in ServiceNow (INC, RITM, SCTASK…)
  2. Click Export Ticket in your bookmarks bar
  3. A blue "Export Ticket" button appears on the page
  4. Click that button — the JSON is copied to the clipboard
  5. Go to Form in Agent Eval and paste the JSON
3

Paste in Agent Eval

On the Form page, use the "Paste ServiceNow" button or paste directly into the import field. The form fills in automatically.

Example of exported data
{
  "number": "INC0435166",
  "shortDescription": "Cannot access shared drive",
  "assignedTo": "John Smith",
  "assignmentGroup": "ITEC-SUP PS",

  "slaBreach": "No",

}

Troubleshooting

The "Export Ticket" button doesn't appear in ServiceNow

Make sure you're inside an open ticket. The script needs g_form to be available on the page. It also works with ServiceNow's classic view.

slaPercentage shows as null

Expand the "SLA" / "Related Lists" section on the ticket before clicking Export Ticket. ServiceNow loads SLAs lazily.

Error: "g_form not available"

This message appears if you click the bookmark outside a ticket (e.g. on the incident list). Open a specific ticket and try again.

I can't drag the bookmark (Safari / Firefox)

Click "Copy bookmarklet code", then create a new bookmark manually and paste the code as the bookmark URL.
